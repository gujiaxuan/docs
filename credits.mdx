## How Credits Work

Each API request consumes credits based on the model used and the number of tokens processed. Different models have different credit multipliers.

### Credit Multipliers

Credits are calculated from token usage with per-model multipliers:

```
credits_used = (input_tokens × input_multiplier) + (output_tokens × output_multiplier)
```

| Model | Input Multiplier | Output Multiplier |
|-------|-----------------|-------------------|
| GPT-4o | 1× | 3× |
| GPT-4o mini | 0.1× | 0.4× |
| Claude Sonnet 4 | 1× | 5× |
| Claude Haiku 3.5 | 0.25× | 1.25× |
| Gemini 2.5 Pro | 1.25× | 10× |
| Gemini 2.5 Flash | 0.15× | 0.6× |
| DeepSeek Chat | 0.14× | 0.28× |
| DeepSeek Reasoner | 0.55× | 2.19× |

<Info>
  Multipliers are subject to change as provider pricing evolves. Check the [Models](/models) page or the dashboard for the latest rates.
</Info>

### Example Calculation

Using Claude Sonnet 4 with 1,000 input tokens and 500 output tokens:

```
credits = (1000 × 1) + (500 × 5) = 1000 + 2500 = 3,500 credits
```

## Daily Credit Refresh

<Note>
  Credits refresh daily at **UTC 00:00** (midnight UTC).
</Note>

Key points about the credit system:

- **Daily accumulation**: Unused credits roll over and accumulate — they do not reset to zero each day. Your balance grows by your plan's daily allowance every day.
- **Refresh time**: New credits are added at UTC 00:00 each day.
- **No overage charges**: When your credits run out, requests return a `429` error until the next refresh.
- **Plan upgrades**: Take effect immediately with prorated credits.

## FAQ

<AccordionGroup>
  <Accordion title="What happens when I run out of credits?">
    Your API requests will return a `429 Too Many Requests` error. Credits will be replenished at the next UTC 00:00 refresh.
  </Accordion>
  <Accordion title="Can I buy additional credits?">
    Currently, credits are included with your subscription plan only. Upgrade your plan for a higher daily allowance.
  </Accordion>
  <Accordion title="Do unused credits expire?">
    Credits accumulate daily and do not expire as long as your subscription is active. They do not reset — they roll over.
  </Accordion>
  <Accordion title="How do I check my remaining credits?">
    Visit the [Dashboard](https://hapuppy.com/dashboard) to view your current balance and usage history.
  </Accordion>
</AccordionGroup>
