<Steps>

<Step title="Create an account">
  Sign up for a free account at [hapuppy.com/register](https://hapuppy.com/register).
</Step>

<Step title="Choose a plan">
  Select a subscription plan that fits your needs:

  | Plan | Price | Daily Credits | Rate Limit |
  |------|-------|---------------|------------|
  | Lite | $9.99/mo | 1M | 100 RPM |
  | Standard | $19.99/mo | 4M | 100 RPM |
  | Pro | $36.99/mo | 8M | 100 RPM |

  Subscribe through the [dashboard](https://hapuppy.com/dashboard). Payments are processed via Creem.
</Step>

<Step title="Generate an API key">
  Navigate to **Dashboard → API Keys** and click **Create Key**. Your key will look like:

  ```
  sk-hapuppy-xxxxxxxxxxxxxxxxxxxxx
  ```

  <Warning>
    Store your API key securely. Do not share it or commit it to version control.
  </Warning>
</Step>

<Step title="Send your first request">
  Use any OpenAI-compatible SDK or make a direct HTTP request.

  <CodeGroup>

  ```python Python
  from openai import OpenAI

  client = OpenAI(
      api_key="sk-hapuppy-xxx",
      base_url="https://api.hapuppy.com/v1"
  )

  response = client.chat.completions.create(
      model="claude-sonnet-4-20250514",
      messages=[
          {"role": "user", "content": "What is the meaning of life?"}
      ]
  )

  print(response.choices[0].message.content)
  ```

  ```typescript TypeScript
  import OpenAI from "openai";

  const client = new OpenAI({
    apiKey: "sk-hapuppy-xxx",
    baseURL: "https://api.hapuppy.com/v1",
  });

  const response = await client.chat.completions.create({
    model: "claude-sonnet-4-20250514",
    messages: [
      { role: "user", content: "What is the meaning of life?" }
    ],
  });

  console.log(response.choices[0].message.content);
  ```

  ```bash cURL
  curl https://api.hapuppy.com/v1/chat/completions \
    -H "Authorization: Bearer sk-hapuppy-xxx" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "claude-sonnet-4-20250514",
      "messages": [
        {"role": "user", "content": "What is the meaning of life?"}
      ]
    }'
  ```

  </CodeGroup>
</Step>

<Step title="Try different models">
  Swap the `model` parameter to try any of our 100+ supported models:

  ```python Python
  # Try GPT-4o
  response = client.chat.completions.create(
      model="gpt-4o",
      messages=[{"role": "user", "content": "Hello!"}]
  )

  # Try Gemini
  response = client.chat.completions.create(
      model="gemini-2.5-pro-preview-05-06",
      messages=[{"role": "user", "content": "Hello!"}]
  )

  # Try DeepSeek
  response = client.chat.completions.create(
      model="deepseek-chat",
      messages=[{"role": "user", "content": "Hello!"}]
  )
  ```

  See the full list of available models on the [Models](https://hapuppy.com/models) page.
</Step>

</Steps>
