<Accordion title="What is Hapuppy?">
  Hapuppy is an AI API gateway that lets you access 100+ AI models (Claude, GPT, Gemini, Grok, DeepSeek, Mistral, Llama, GLM, and more) through a single API key. It's fully compatible with the OpenAI API format, so it works as a drop-in replacement in any tool or SDK that supports OpenAI.
</Accordion>

<Accordion title="How is Hapuppy different from using provider APIs directly?">
  - **One API key** for all models — no need to manage separate accounts and keys for each provider
  - **Flat-rate pricing** — predictable monthly cost instead of per-token billing
  - **OpenAI-compatible** — works with any tool or SDK that supports the OpenAI format
  - **Easy model switching** — change the `model` parameter to try different models instantly
</Accordion>

<Accordion title="Which models are supported?">
  Hapuppy supports 100+ models from providers including Anthropic (Claude), OpenAI (GPT), Google (Gemini), xAI (Grok), DeepSeek, Mistral, Meta (Llama), and Zhipu (GLM). See the [Models](https://hapuppy.com/models) page for the full list.
</Accordion>

<Accordion title="Is Hapuppy compatible with the OpenAI SDK?">
  Yes. Hapuppy is a drop-in replacement for the OpenAI API. Simply set your base URL to `https://beta.hapuppy.com/v1` and use your Hapuppy API key. Both the Python and TypeScript OpenAI SDKs are fully supported.
</Accordion>

<Accordion title="What tools does Hapuppy work with?">
  Hapuppy works with any tool that supports the OpenAI API format, including:
  - **Claude Code** — Anthropic's CLI tool
  - **Cursor** — AI code editor
  - **Cline** — VS Code AI assistant
  - **Continue** — Open-source AI code assistant

  See the [Integrations](/integrations) page for setup instructions.
</Accordion>

<Accordion title="How does pricing work?">
  Hapuppy offers three monthly subscription plans:
  - **Lite** — $9.99/mo with 1M daily credits
  - **Standard** — $19.99/mo with 4M daily credits
  - **Pro** — $36.99/mo with 8M daily credits

  All plans include access to every model. Credits are consumed based on token usage with per-model multipliers. See [Credits](/credits) for details.
</Accordion>

<Accordion title="Do unused credits roll over?">
  Yes. Credits accumulate daily and do not reset. Your balance grows by your plan's daily allowance every day at midnight Beijing time (UTC 16:00).
</Accordion>

<Accordion title="What happens when I run out of credits?">
  API requests will return a `429` error. Credits refresh at the next midnight Beijing time (UTC 16:00). You can also upgrade your plan for a higher daily allowance.
</Accordion>

<Accordion title="What is the rate limit?">
  All plans have a rate limit of 100 requests per minute (RPM). If you exceed this limit, you'll receive a `429` error.
</Accordion>

<Accordion title="Does Hapuppy support streaming?">
  Yes. Set `stream: true` in your request to receive tokens as they are generated. See the [Chat Completions](/chat-completions#streaming) docs for examples.
</Accordion>

<Accordion title="Does Hapuppy support function calling?">
  Yes. Function calling (tool use) is supported for models that natively support it, such as GPT-4o, Claude Sonnet/Opus, and Gemini Pro. See [Chat Completions](/chat-completions#function-calling) for examples.
</Accordion>

<Accordion title="Does Hapuppy support vision/image inputs?">
  Yes. For models that support multimodal input (GPT-4o, Claude Sonnet, Gemini), you can send images in your messages. See [Chat Completions](/chat-completions#vision) for details.
</Accordion>

<Accordion title="How do I get an API key?">
  1. Sign up at [hapuppy.com/register](https://hapuppy.com/register)
  2. Subscribe to a plan
  3. Generate an API key from the dashboard

  See the [Quickstart](/quickstart) guide for step-by-step instructions.
</Accordion>

<Accordion title="What payment methods are accepted?">
  Payments are processed via Creem. Credit/debit cards are accepted.
</Accordion>

<Accordion title="How do I cancel my subscription?">
  You can cancel your subscription at any time from the [Dashboard](https://hapuppy.com/dashboard). Your access continues until the end of the current billing period.
</Accordion>

<Accordion title="How do I contact support?">
  Email us at [support@hapuppy.com](mailto:support@hapuppy.com).
</Accordion>
